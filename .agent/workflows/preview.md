---
description: Автоматична збірка та візуальна перевірка змін
---

Цей воркфлоу дозволяє автоматизувати процес розробки:

1. Запустіть `dev.py` у терміналі (якщо він ще не запущений):
// turbo

```bash
python dev.py
```

1. Кожного разу, коли ви змінюєте CSS чи JS у папці `frontend/`, скрипт автоматично оновлює папку `site/`.

2. Для візуальної перевірки змін я (Antigravity) можу використовувати інструмент браузера:
   - Перейти на <http://localhost:8080>
   - Ввести пароль (якщо потрібно)
   - Зробити скріншот або перевірити конкретні елементи

3. Якщо ви хочете, щоб я перевірив конкретну зміну, просто скажіть: "Перевір як виглядає кнопка/меню".
